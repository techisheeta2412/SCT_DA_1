# Sales Performance Dashboard

## SkillCraft Technology - Data Analytics Task 1

### Project Overview
This project presents an interactive Sales Performance Dashboard created using Microsoft Excel. The dashboard is designed to analyze sales data and provide clear insights into regional sales, category-wise performance, and sales trends over time.

### Dashboard Features
- Total Sales by Region
- Sales by Category
- Sales Trend Over Time
- Interactive Region Slicer
- Interactive Category Slicer
- PivotTables and PivotCharts
- Conditional Formatting for performance analysis

### Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Conditional Formatting

### Key Insights
- The West region recorded the highest overall sales.
- Technology was the highest-performing product category.
- Sales performance showed fluctuations over time with noticeable growth during later periods.
